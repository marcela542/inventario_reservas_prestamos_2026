# Sistema de Inventario

Aplicación web para la gestión de inventario, reservas y préstamos de elementos electrónicos.

## Instalación

1. Clonar el repositorio
2. Crear una base de datos en MySQL (ej: inventario)
3. Importar el archivo:
   /database/inventario.sql
4. Configurar la conexión en:

   Lib/conf.php

## Configuración BD

Por defecto el proyecto usa:

Host: localhost  
Usuario: root  
Contraseña: (vacía)  
Base de datos: inventario  

Si tu entorno es diferente, modifica el archivo:
Lib/conf.php

## Tecnologías

- PHP (MVC)
- MySQL
- JavaScript
- HTML / CSS

## Notas

Este proyecto está configurado para entorno local (XAMPP o similar).
